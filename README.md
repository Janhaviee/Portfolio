# Personal Portfolio Website

## About the Project

This is my personal portfolio website created using HTML and CSS. The website showcases my skills, projects, education, and contact information in a clean and responsive design.

## Features

* Responsive design for mobile, tablet, and desktop devices
* Navigation bar
* Home section
* About section
* Services section
* Contact section
* Smooth scrolling and hover effects

## Technologies Used

* HTML5
* CSS3
* Visual Studio Code
* Git & GitHub

## Purpose

The purpose of this project is to present my profile, skills, and projects while practicing frontend web development concepts and responsive web design.
